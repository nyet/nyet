Here's what I've found today. To add a description, use the "Image Description" tag (plain "description" doesn't do it), newlines are okay:

exiftool -ImageDescription='text here' IMAGE

To add flickr tags, use one "keywords" tag each:

exiftool -keywords="Multi Word Keyword" -keywords="another keyword" IMAGE

If the keywords contain double-quotes ("), flickr will strip them out.

The Model tag gets used to populate the "Taken with " info:

exiftool -Model="My Special Camera 17MP"

Of the eight or so date tags, just plain "Date" seems to be the one to use:

exiftool -Date="2006:04:16 12:53:24" IMAGE
